### Description
**Drowning in browser tabs?** TabPilot is the autopilot for your browser — it automatically organizes tabs into smart groups and lets you search through them instantly.

🗂️ **One-Click Smart Grouping**
- Automatically categorizes tabs into groups: Work, Development, Learning, Social, Entertainment, Shopping, Docs, and more
- Uses a built-in rule engine with 100+ domain rules and keyword patterns
- Works instantly with zero configuration — no AI API key needed
- Creates native Chrome Tab Groups with color coding

🔍 **Lightning-Fast Tab Search**
- Search across all open tabs and recently closed tabs
- Fuzzy matching on titles and URLs
- Keyboard shortcut (Ctrl+Shift+F) to open search instantly
- Results highlighted with relevance scoring

🤖 **Optional AI Enhancement**
- Bring your own API key (OpenAI, Claude, or any OpenAI-compatible endpoint)
- AI handles the tabs that rule-based classification can't categorize
- Boosts classification accuracy from ~75% to ~95%
- Your API key stays on your device — we never see it

🔒 **Privacy First**
- All data stored locally on your device
- No accounts, no tracking, no analytics
- Zero data sent to our servers
- Open source and transparent

🌐 **Bilingual**
- Full English and Chinese (简体中文) support
- Automatically matches your Chrome language

**Perfect for:**
- Developers with 50+ tabs open
- Researchers managing multiple topics
- Anyone who wants a cleaner browser

### 描述
**标签页太多？** TabPilot 是你的标签页自动驾驶仪 — 自动整理标签页为智能分组，支持即时搜索。

🗂️ **一键智能分组**
- 自动将标签分为：工作、开发、学习、社交、娱乐、购物、文档等分组
- 内置规则引擎，包含 100+ 域名规则和关键词匹配
- 开箱即用，零配置 — 不需要 AI API 密钥
- 使用 Chrome 原生标签组，带颜色标识

🔍 **闪电搜索**
- 搜索所有打开的标签和最近关闭的标签
- 标题和 URL 模糊匹配
- 快捷键 (Ctrl+Shift+F) 即时打开搜索
- 结果高亮显示，按相关性排序

🤖 **可选 AI 增强**
- 自带 API 密钥（支持 OpenAI、Claude 或任何兼容接口）
- AI 处理规则引擎无法分类的标签
- 分类准确率从 ~75% 提升到 ~95%
- API 密钥仅存储在本地 — 我们永远看不到

🔒 **隐私优先**
- 所有数据存储在本地
- 无账号、无追踪、无分析
- 零数据发送到我们的服务器
